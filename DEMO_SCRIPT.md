# Demo Script

## Goal

Deliver a clean 3-5 minute walkthrough that shows the app is polished, useful, and different from a basic CRUD finance tracker.

## Demo Flow

### 1. Open Dashboard

- Show summary cards
- Point out spending analytics
- Show budget status and recent transactions
- Mention that AI insights are based on real app data, not a generic chatbot

### 2. Add or review a transaction

- Go to Transactions
- Add one expense or income item
- Return to Dashboard briefly to show data updates

### 3. Show Budgets

- Open Budgets
- Explain how monthly category limits work
- Point out overspending / warning behavior if available

### 4. Show Reports

- Open Reports
- Change the report range
- Show spending by category and monthly comparison
- Click `Export CSV` to show that data can be taken out of the system

### 5. Show AI Insights Chat

- Ask:
  - “If I want to save 5 million this month, how much am I missing?”
  - “Which category should I reduce first?”
- Explain that the answer is grounded in the user’s real transactions and budgets

## Talking Points

- This is not only a finance logger, it also supports budgeting, analytics, and guided decisions
- The AI feature uses real personal finance data from the app
- Reports can be exported for review or submission
- The UI was polished to feel like a product, not only a technical prototype

## Backup Plan

If live edits fail during demo:

- Use an existing seeded/demo account
- Focus on dashboard, reports, AI, and export
- Avoid risky live registration flows if internet/email delivery is unstable
